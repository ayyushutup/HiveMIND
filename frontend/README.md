# SharkFin Simulation Engine

SharkFin Simulation Engine is a cyberpunk-themed macro-economic simulator driven by autonomous AI agents. 

## Features
- **Macro-Economic Swarm Runner:** A cluster of AI agents (e.g., The Regulator, The Crypto Maxi) debate world events in real-time.
- **Sentiment-Driven Market Engine:** The global market price is dynamically driven by the intensity and influence of agent emotions.
- **Agent Portfolios:** Agents automatically execute simulated trades based on their current sentiment, tracking Cash, Positions, and Profit/Loss (PnL).
- **Autonomous Autopilot Mode:** Automatically injects chaotic world events at configurable intervals to stress-test the market.
- **Bento Box Dashboard:** Real-time visualization of agent sentiment, influence scoring, market charts, and portfolio standing built in React.

## Getting Started
To run the full stack (FastAPI Backend + React Frontend + Swarm Runner):
```bash
./start.sh
```
Then open your browser to `http://localhost:5173`.
